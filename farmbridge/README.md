# 🌾 FarmBridge – Farmers' Well-being Platform

FarmBridge is a web application designed for **Indian farmers** to access crucial information such as:
- **Government Schemes**
- **Market Prices**
- **Weather Updates**
- **AI-based Crop Suggestions**  
... all in one place with support for **Hindi & English**.

---

## 🚀 Features
- **Multi-language Support (Hindi & English)**  
- **AI-based Crop Recommendation** based on real-time weather conditions  
- **Market Prices** with trending crops and full rate table  
- **Government Schemes Section** for awareness  
- **Weather Info Section** using **OpenWeather API**  
- **Search Bar** to find crop data quickly  
- **Responsive UI** for mobile & desktop  

---

## 🛠️ Tech Stack
- **Frontend**: React.js  
- **Styling**: CSS3 (Responsive Design)  
- **API**: OpenWeatherMap API (for weather data)  
- **Language Support**: i18n-based translations (Hindi & English)
